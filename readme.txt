# Loan Calculator
Laon calculator fetch data from API and calculate loan according to provided amount. 

## Instructions

`git clone `

`npm install`

`npm run start` & Enjoy the new look in your browser.

## Tools used

* React 16.7
* React Router Dom 4.3.x
* Bootstrap 4
* React Bootstrap 0.32
* React Axios
* Babel core 7
* Babel loader 8
* Webpack 4
* Sass Dependencies ([css|file|image-webpack|raw|sass|style|url] Loader)
